## About the Project

**StudyVault** is a web-based student loan marketplace designed to create a secure and transparent ecosystem for education financing. The platform brings together **students seeking educational loans** and **lenders willing to provide financial assistance** on a single digital platform.

The primary objective of StudyVault is to reduce loan fraud through multiple layers of digital identity verification and AI-assisted trust evaluation.

## Problem Statement

Traditional education loan processes often involve:

* Lengthy paperwork
* Manual verification
* Identity fraud risks
* Lack of transparency between borrowers and lenders

StudyVault aims to simplify and secure this process through digital verification mechanisms.

## Key Features

### Student Registration

* Secure account creation
* Login and authentication interface

### Education Loan Application

* Multi-step loan application form
* Personal and academic information collection
* Document upload support

### Document Verification (Concept)

* OCR-based extraction of document information
* Automatic comparison between uploaded documents and user-entered details

### Video KYC (Concept)

* Video verification interface
* Future support for facial recognition matching

### Safety Score System

A unique trust metric generated after verification checks, allowing lenders to assess the authenticity of student profiles.

### Student Dashboard

* Application tracking
* Verification status monitoring
* EMI calculator
* AI verification log simulation

### AI Support Chatbot

* Interactive assistance
* Common loan and verification FAQs
* Quick response interface

## How to Run

1. Clone the repository

```bash
git clone https://github.com/yourusername/StudyVault.git
```

2. Open the project folder.

3. Run the terminal or the Command Prompt in the project folder.

4. Type and run the command:

```bash
    python -m htto.server 8000
```
This will start the web server and you can view the project in your browser at `http://localhost:8000`.
